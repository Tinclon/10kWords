# 10kWords

Multilingual verb conjugation reference. All moods, all tenses, all irregular patterns.

**[View the site](https://tinclon.github.io/10kWords/10kWords.html)**

## Languages

| Language | Family | Moods | Tenses |
|----------|--------|-------|--------|
| English | Germanic | Indicative, Subjunctive, Conditional, Imperative | 16 |
| French | Romance | Indicative, Subjunctive, Conditional, Imperative | 16 |
| Spanish | Romance | Indicative, Subjunctive, Conditional, Imperative | 16 |
| Portuguese | Romance | Indicative, Subjunctive, Conditional, Imperative, Personal Infinitive | 20 |
| Czech | Slavic | Indicative, Conditional, Imperative | 6 |

## Features

- Irregular verb patterns grouped by conjugation class
- Regular verb examples at the top of each column for reference
- Dimmed patterns indicate verbs that are regular in that tense
- Collapsible additional info for key grammar distinctions (ser vs. estar, etre vs. avoir, etc.)
- Responsive layout with column selector on small screens
- Color-coded per language
